Business Manager Prototype Hub

A collaborative prototype environment for Product Managers to experiment with UX updates and design changes. Click any screen below to explore the Salesforce Developer Workspace design.

🚀 Getting Started

This prototype uses the Salesforce Developer Workspace design system with dual sidebar navigation, cloud branding, and consistent styling. All screens are built with pure HTML/CSS/JS for easy collaboration with Claude Code.

Live Demo: lukejohnson-sf.github.io/PrototypeBM
Repository: github.com/lukejohnson-sf/PrototypeBM

For Product Managers

Step 1: Create your branch: git checkout -b pm-yourname

Step 2: Navigate to any screen above and explore the interface

Step 3: Use Claude Code to modify screens or create new ones

Step 4: Test changes by opening HTML files in your browser

Step 5: Commit and push changes to share with the team

Example Claude Prompts:

  • 💬 "Add a notification bell icon to the header with a badge count"
  • 💬 "Create a new screen for site preferences with tabs"
  • 💬 "Update the products grid to show 4 columns instead of auto-fit"
  • 💬 "Change the primary blue to match our new brand color #0070D2"
