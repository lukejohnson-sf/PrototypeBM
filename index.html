How to Use This Prototype

For Product Managers:

  1. Create your own branch: git checkout -b pm-yourname
  2. Navigate to any screen above and explore the interface
  3. Use Claude Code to modify screens or create new ones
  4. Test your changes by opening HTML files in your browser
  5. Commit and push your changes to share with the team

Example Claude Prompts:

  • "Add a filter bar to the products page"
  • "Create a new screen for site preferences"
  • "Update the dashboard to show revenue trends"
  • "Change the primary color to #FF6B35"

Project Information

Repository: bm-prototype

Branch Strategy: Each PM works on their own branch (pm-name), merges to main when ready

Tech Stack: Pure HTML/CSS/JavaScript (no build process required)

Design System: SFCC Business Manager style guidelines
